# float!

`float!`是*IEEE-754*标准的64位双精度浮点数。`float64!`是它的别名。

## 格式

```
<sign><digits>.<digits>
<sign><digits>E<exponent>
<sign><digits>.<digits>E<exponent>
```
其中：
* <sign>     : 正负号 `+|-`,可选。
* <digits>   : 数位。
* <exponent> : 带符号整数。

示例：

```
0.0
1.0
-12345.6789
3.14159265358979
-1E3
+1.23456E-265
```

字面值最多接受16位数字。多出的部分不显示。

## 值型转换

`float!`可以转换为`float32!`。

```
pi: 3.14159265358979
pi-32: as float32! pi
print pi-32
;输出3.1415927
```

## 数学运算

* 支持Red/System的全部数学运算符：`+, -, *, /, //, %`。
* 默认舍入方法为`就近舍入`。
* 两边的操作数都必须是`float!`，没有隐式转换。
* 取模`//`和取余`%`的结果相同。
